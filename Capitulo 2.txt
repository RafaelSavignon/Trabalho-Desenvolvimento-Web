Respostas:

Exercício 2.1:
O elemento HTML que contém todos os demais elementos de uma página é o <html>. Ele representa a raiz do documento HTML e engloba todos os elementos presentes na página.

Exercício 2.2:
O elemento HTML que contém elementos com metainformações sobre uma página web é o <head>. Este elemento inclui informações como título, metatags, links para estilos CSS, scripts, entre outros.

Exercício 2.3:
O elemento HTML que contém todos os elementos de conteúdo de uma página web é o <body>. Este elemento engloba o conteúdo principal da página, como texto, imagens, links, formulários, etc.

Exercício 2.4:
O elemento HTML que permite associar um documento CSS a uma página web na seção de metainformações é o <link>. Usado dentro do elemento <head>, o <link> estabelece a ligação entre a página HTML e o arquivo CSS, influenciando a apresentação visual da página.
